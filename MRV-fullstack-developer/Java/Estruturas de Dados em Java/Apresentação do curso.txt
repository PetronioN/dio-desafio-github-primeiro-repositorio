## Tipos de Memória

- Memória Primária: mantém os dados armazenados somente
na presença de energia.

- Memória Secundária: mantém os dados mesmo na
ausência de energia.

## Os dados na memória

A memória armazena dados somente com 0 e 1.

0001 = 1
0010 = 2
0011 = 3
0100 = 4
0101 = 5
0110 = 6
0111 = 7
1000 = 8
1001 = 9
1010 = 10
