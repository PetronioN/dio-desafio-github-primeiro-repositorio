## O que é JSX

const element = <h1>Hello, world!</h1>;

Nem HTML e nem uma string!

- React não separa as tecnologias colocando marcação
em arquivos separados, e sim separa conceitos e mantém
o código pouco acoplado chamando-os de componentes.

- Não é obrigatório a utilização do JSX.

- Sintax Suggar para React.createElement...

- O browser não interpreta o JSX, para isso é necessário
um transpilador para que a aplicação possa entender o código.
O mais utilizado é o Babel.
